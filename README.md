# Ni

A more thorough (and extensible) npm init.

## Installation

```
$ git clone https://github.com/pacocoursey/ni.git
$ sudo npm install -g
```

## Usage

Run `ni` in any folder you wish to initialize a new project.

```
$ ni
```
